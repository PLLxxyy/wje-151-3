# 温馨家政 - 家政服务预约平台

纯前端家政服务预约平台，所有数据存储在浏览器 localStorage 中，无需后端服务。

## 技术栈

- **构建工具**: Vite 5
- **前端框架**: React 18 + TypeScript
- **路由**: React Router 6 (Hash Router)
- **数据持久化**: localStorage
- **无后端依赖**

## 启动方式

```bash
npm install
npm run dev
```

启动后访问 http://localhost:5173

## 功能说明

### 用户端

- **服务分类首页**: 保洁、维修、搬家、做饭四大服务分类入口，热门推荐
- **服务列表**: 按分类浏览服务，支持按价格升序/降序、评分排序
- **服务详情**: 查看服务项目明细、所需时长、价格明细、阿姨介绍
- **预约下单**: 选择日期时段、填写地址和备注，支持常用地址一键选用
- **我的订单**: 订单列表按时间倒序，支持按状态筛选（全部/待确认/服务中/已完成）
- **评价**: 已完成订单可给阿姨评分(1-5星)和文字评价
- **常用地址**: 添加、编辑、删除、设置默认地址，下单时可快速选用

### 阿姨端

- **工作台**: 待接订单列表，一键接单
- **服务管理**: 查看进行中的服务，标记完成
- **评价查看**: 查看用户给的评分和文字评价
- **角色切换**: 用户/阿姨视角随时切换

### 数据说明

首次加载自动 seed 12 个服务、8 位阿姨、4 条示例订单、2 条常用地址、2 条示例评价。支持在个人中心清除数据重新加载。
